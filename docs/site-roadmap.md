# Site Roadmap

## Purpose

`pieryudg.github.io` is the public front door for Pieryudg projects, tools, documentation, and future service links.

## Near Term

- Keep the homepage simple and useful.
- Highlight `Data Plot Chart Analyzer` as the first project.
- Add a compact project index.
- Add a clear place for future NAS or self-hosted service links without exposing private infrastructure too early.
- Add a changelog for site updates.

## Medium Term

- Add a project page for Data Plot Chart Analyzer with screenshots, download notes, and examples.
- Add a software/tools page for engineering utilities.
- Add a private-services placeholder page explaining how access works without revealing private endpoints.
- Add a custom domain only after the public/private boundary is clear.

## Operating Principle

Use Cursor to decide what should exist. Use VS Code/Codex to build it, test it, and publish it.
